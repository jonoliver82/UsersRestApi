## Users REST API Sample

This project shows examples of the following concepts:

* Entity Framework Core 2.2 InMemory database for an ASP .NET Core Web API
* Value Objects with Entities
* Use of Specification and Query objects to encapsulate predicate and selection expressions when performing a query to a Repository
